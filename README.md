# Antoine Ludeau Portfolio

Welcome to my portfolio source code repository. Don't hesitate to clone the repo and start it.

## Techno

My portoflio uses a number of open source projects to work properly:
- [ReactJS](https://reactjs.org/) - A JavaScript library for building user interfaces!
- [TypeScript](https://www.typescriptlang.org/) - JavaScript with syntax for types!
- [MaterialUI](https://mui.com/) - A React component library! 

## Installation

```sh
npm i
```

# Development

```sh
npm start
```

# Production

```sh
npm run build
```
